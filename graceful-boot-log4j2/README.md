# 日志处理模块

## 一、说明

日志实现，通过引入本模块，直接获得log4j2异步日志处理的能力

## 二、核心功能

- 基于disruptor的异步日志

- dev和test环境输出控制台，生产环境不输出控制台

- 提供默认日志级别、存储路径，减少开发配置，并支持自定义日志级别、存储路径

## 三、组件版本

| 组件         | 版本     |
|------------|--------|
| log4j2     | 2.21.1 |
| disruptor  | 3.4.4  |
| springboot | 3.2.5  |

## 四、快速入门

step1: 直接引入模块依赖。

step2: 把其他日志框架排除掉，例如logback。

step3: 根据需要配置路径和日志级别

默认的日志配置如下：

```yaml
logging:
  level:
    cm.fw: DEBUG
    root: INFO
```
